## 定向拉流（回源）

定向拉流功能是向配置中的固定主机地址拉流，分为两种：1.按需拉流（回源），2.静态拉流（回源）
{:.success}
^

- 按需拉流（回源）

服务器只在收到play消息时检查服务器是否已经存在该流，如果不存在则向指定主机地址拉流。
{:.info}
^

```nginx
rtmp {
    server {
        listen 1935;
        application live {
            live on;
            pull rtmp://xxx/xxx;
        }
    }
}
```

- 静态拉流（回源）

服务器在进程创建后就立即向指定主机地址拉流。
{:.info}
^

```nginx
rtmp {
    server {
        listen 1935;
        application live {
            live on;
            static_pull rtmp://xxx/xxx/xxx;
        }
    }
}
```
